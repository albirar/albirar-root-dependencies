Workflow and Git

All Albirar projects uses the Gitflow workflow as strategy to manage the development process.

This parent pom includes a fully configured jgit-flow-plugin to enforce the use of git workflow through maven.

New feature

  • mvn jgitflow:feature-start
  • Coding and commiting
  • Testing
  • mvn jgitflow:feature-finish

New hotfix

  • mvn jgitflow:hotfix-start
  • Coding and commiting
  • Testing
  • mvn jgitflow:hotfix-finish

New release

  • mvn jgitflow:release-start
  • Testing, amending and commit
  • QA if needed
  • mvn jgitflow:release-finish
